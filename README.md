# Kronos 博客系统

# 安装运行
### golang Install
```bash
brew install go
```
### 运行博客 
```bash
go mod download
go mod vendor
```
### 编译博客 && Make
```bash
build 编译项目
test 执行单元测试
clean 清除编译缓存
run 运行项目
restart 重启
mod 安装项目依赖
cross 跨平台编译
fmt 格式化代码
```
